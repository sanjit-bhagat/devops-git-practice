# Git Basics Practice

## Overview
This repository contains my hands-on practice of basic Git commands. The purpose of this repository is to learn and understand Git version control concepts through practical exercises.

## Topics Practiced

- Git Installation Verification
- Git Configuration
  - git config --global user.name
  - git config --global user.email

- Repository Initialization
  - git init

- File Tracking
  - git status
  - git add
  - git add .

- Commit Operations
  - git commit -m "message"

- Branch Management
  - git branch
  - git branch <branch-name>
  - git checkout <branch-name>
  - git switch <branch-name>

- Viewing History
  - git log
  - git log --oneline

## Practice Objective

The goal of this practice is to:

- Understand Git workflow
- Track file changes efficiently
- Manage branches
- Create meaningful commits
- Connect local repositories with GitHub
- Build a strong foundation for DevOps learning
